Bienvenido a la API de F1 2025

Esta es una API de solo lectura que proporciona información sobre el campeonato mundial de pilotos de la temporada pasada.

Endpoints disponibles:

GET /api/datos

Descripción: Devuelve una lista completa en formato JSON con los pilotos, sus puntos acumulados, la escudería a la que pertenecen y un valor booleano que indica si lograron al menos una pole position durante la temporada.
